# Dotfiles

## Naming convention
~/.config/$PROGRAM // Real path  
~/dotfiles/$PROGRAM/.config/$PROGRAM // Dotfile repo path  

## USAGE
1. cd dotfiles/  
2. stow $PROGRAM/  
### Alternatively
2. "stow \*" // If all configs should be stowed

## Dependencies/Version stuff    
gnu stow  
neovim 0.7.0   
node v18.7.0
